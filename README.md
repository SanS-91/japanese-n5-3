# 🇯🇵 Japanese Learning Path

Học tiếng Nhật từ N5 đến N3 - 30 phút mỗi ngày

## 🌐 Demo

Truy cập: https://YOUR_USERNAME.github.io/japanese-learning/

## 📚 Lộ trình

- **N5** (24 tuần): Nhập môn - Bảng chữ cái, từ vựng cơ bản
- **N4** (26 tuần): Sơ cấp - Ngữ pháp trung cấp, hội thoại
- **N3** (30 tuần): Trung cấp - Kính ngữ, đọc hiểu, nghe hiểu

## ✨ Tính năng

- 🔊 Phát âm chuẩn (Text-to-Speech)
- 🎴 Flashcard với Spaced Repetition
- 📝 Bài tập tương tác
- 📊 Theo dõi tiến độ
- 🌙 Dark mode
- 📱 Responsive design

## 🛠️ Công nghệ

- HTML5, CSS3, JavaScript
- Web Speech API (Text-to-Speech)
- LocalStorage (lưu tiến độ)

## 📖 Cách sử dụng

1. Mở `index.html` để xem lộ trình
2. Chọn cấp độ (N5/N4/N3)
3. Học theo thứ tự từng tuần
4. Mỗi ngày 30 phút, duy trì streak!
